### Feign

- springboot启动类加上@EnableFeignClients

- 新建service包自定义一个service接口

- ```java
  @FeignClient(name = "service-provider")
  public interface EchoService {
      @GetMapping("/test")
      public User getEcho();
  }
  // 其中getmapping对应的服务端api
  //注意，若服务端带有路径变量，客户端的getmapping中必须携带参数
@GetMapping("/test/{name}")
  public User getEcho(@PathVariable String name);
  ```
  
- 在Controller中注入service即可
